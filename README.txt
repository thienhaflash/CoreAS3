AS3 Core library

A library of suplemental classes and package functions for Actionscript 3 which mostly :

1. Shortcuts for common tasks like removeChildren, removeThis, addChildren, tint, formatText, setMouse, flashvar ...
2. Utils like drawArc, drawRect, bitmapShot ...
3. Managers like Enterframe, Aim, Liquid, Drag, Uploader, Downloader ...
4. Quick, one class UI : Input, Scroller, Form, Menu, Media, Progress, UserManager (login / logout / register / forgot pass), Pager, Pan, Swapview, Tabs, KTimer, Button ...
